# G3SEARCH Properties

## Overview
The **G3SEARCH** object provides several properties to configure the indexing and search behavior. These properties define the locations for index storage and document source, as well as file filtering options.

## Properties Summary

| Property | Access | Type | Description |
| :--- | :--- | :--- | :--- |
| **IndexPath** | Read/Write | String | Specifies the file system path where the search index is stored. |
| **DocsPath** | Read/Write | String | Specifies the target directory containing documents to be indexed. |
| **Extension** | Read/Write | String | Specifies the file extension filter used during the indexing process. |
