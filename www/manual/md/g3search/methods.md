# G3SEARCH Methods

## Overview
The **G3SEARCH** object exposes methods for managing the search index and performing full-text queries against indexed content.

## Methods Summary

| Method | Returns | Description |
| :--- | :--- | :--- |
| **BuildIndex** | Empty | Recursively scans the document path and builds/updates the search index. |
| **Search** | Array | Performs a full-text search and returns an array of matching document paths. |
